# 성적 데이터 생성 및 분석 프로젝트 가이드

이 문서는 중학교 1학년 기술교과 성적 데이터를 생성하고, 이를 분석 및 시각화하는 파이썬 코드 사용법을 설명합니다.

---

## 1. 프로젝트 개요

이 프로젝트는 다음과 같은 두 가지 주요 기능을 수행합니다.

1.  **가상 성적 데이터 생성**: 5개 분반(각 30명)의 기말고사 및 수행평가 점수를 무작위로 생성합니다.
2.  **데이터 분석 및 시각화**: 생성된 데이터를 바탕으로 반별 평균을 비교하고, 점수 분포를 시각화하여 이미지로 저장합니다.

---

## 2. 파일 설명

| 파일명 | 설명 |
| :--- | :--- |
| `07_create_grade_data.py` | 성적 데이터를 생성하여 `grade_data.csv`로 저장하는 기본 스크립트입니다. |
| **`08_analyze_grades.py`** | 데이터 생성부터 분석(Pandas), 시각화(Matplotlib)까지 한 번에 수행하는 통합 스크립트입니다. |
| `grade_data.csv` | 스크립트 실행 시 생성되는 학생들의 성적 데이터 파일입니다. |
| `grade_data_analysis.png` | `08_analyze_grades.py` 실행 시 생성되는 시각화 결과 이미지입니다. |

---

## 3. 사전 준비 (필수 라이브러리 설치)

분석 기능을 사용하기 위해서는 외부 라이브러리 설치가 필요합니다. 터미널(명령 프롬프트)에서 아래 명령어를 입력하세요.

```bash
pip install pandas matplotlib
```

- **pandas**: 엑셀처럼 데이터를 표 형태로 다루기 위한 도구
- **matplotlib**: 데이터를 그래프로 그려주는 시각화 도구

---

## 4. 실행 방법

가장 강력한 기능을 제공하는 `08_analyze_grades.py` 파일을 실행하는 것을 권장합니다.

1.  VS Code에서 `08_analyze_grades.py` 파일을 엽니다.
2.  터미널을 열고 아래 명령어를 입력하여 실행합니다.

```bash
python 08_analyze_grades.py
```

3.  실행이 완료되면 다음 결과물을 확인하세요.
    - **터미널 출력**: 반별 평균, 표준편차 등 통계 요약
    - **`grade_data.csv`**: 엑셀로 열어볼 수 있는 전체 성적표
    - **`grade_data_analysis.png`**: 반별 비교 그래프 이미지

---

## 5. 성적 산출 기준 (로직)

성적은 다음과 같은 기준으로 산출되어 **100점 만점**으로 환산됩니다.

- **기말고사 (60%)**: 100점 만점의 시험 점수에 0.6을 곱함 (최대 60점)
- **수행평가 1 (10%)**: 10점 만점
- **수행평가 2 (10%)**: 10점 만점
- **수행평가 3 (20%)**: 20점 만점

> **수식**: `총점 = (기말고사 × 0.6) + 수행1 + 수행2 + 수행3`

---

## 6. 시각화 결과 해석

생성된 `grade_data_analysis.png` 파일은 두 가지 그래프를 보여줍니다.

1.  **박스 플롯 (왼쪽 그래프)**
    - 각 반의 점수가 어떻게 퍼져 있는지 보여줍니다.
    - 상자 안의 선은 **중앙값**을 의미합니다.
    - 상자의 크기가 작을수록 학생들의 점수 차이가 크지 않다는 뜻입니다.

2.  **막대 그래프 (오른쪽 그래프)**
    - 각 반의 **평균 점수**를 비교합니다.
    - 어느 반의 성적이 전반적으로 높은지 한눈에 알 수 있습니다.
